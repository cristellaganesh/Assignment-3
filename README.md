# Assignment-3

# Cristella's Personal Portfolio

## Overview

In this assignment, I created a personal portfolio based on one of the hobbies and interests I enjoy as part of the Front-end Web Development Module 3 assignment. The aim of this project was to improve the overall layout of the HTML structure that I created in the previous module by applying CSS concepts learned throughout Module 3. The styling choices were made to reflect my personal interest in fashion while keeping the website clean, readable, and easy to navigate.

## Visual Styling Choices

### Colour Scheme

I chose a soft pink background for the website to reflect a creative aesthetic that relates to my interest in fashion while still ensuring good readability against the grey text. I added a light green colour to the navigation bar to match the small touches of green in my pixelated cartoon portrait and to make the navigation links stand out clearly. These colours were chosen to keep the website visually simple while maintaining strong readability and still adding personality and charm.

### Typography

The main font used for headings is a Google Font called **Iosevka Charon Mono**, which gives the website a modern, slightly game-like appearance that matches the theme while still being simple enough for viewers to read comfortably. Paragraph text uses a simple and highly readable font (Arial) so that the content remains easy to follow. Headings are also bold and underlined to help clearly distinguish the different sections of the page.

### Layout and Spacing

Basic layout styling was applied using CSS properties such as **margin**, **padding**, and **text alignment**. These properties helped organise the content and create spacing between elements so that the page does not appear cluttered or difficult to read. The navigation links were styled so that they appear as small buttons rather than plain text links, which improves both navigation and visual structure.

## Validation

The stylesheet was validated using the W3C CSS Validation Service to ensure that the code follows proper CSS standards. Any issues that appeared during validation were corrected to ensure the CSS remained clean and valid.

## Theme and Multimedia Elements

My portfolio follows a creative, feminine, and fashion-inspired theme. I wanted the design to reflect my passion for fashion while still maintaining a clear and structured layout. Because the portfolio introduces my hobbies and personality, the design choices aim to visually represent creativity while remaining simple and easy to understand.

Images were also included to support the theme and provide visual examples related to fashion, inspiration, and design. These multimedia elements help make the page more engaging while reinforcing the theme of fashion and creativity.

Overall, the theme attempts to balance personal creativity with a structured layout, representing both my interest in fashion and my journey into front-end web development.

## Reflection

While working on this assignment, I learned how CSS can transform a basic HTML structure into something more visually appealing and engaging. I find CSS more creative and enjoyable to work with than HTML; however, I understand that both languages are essential and work together to create functional, well-structured websites.

This assignment helped me better understand how design choices such as colour, typography, and layout contribute to the overall look and feel of a website.
